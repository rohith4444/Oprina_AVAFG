# =============================================================================
# Key Dependencies Explanation
# =============================================================================

# google-adk: Agent Development Kit for multi-agent orchestration
# supabase: Database client for conversation context and chat history
# redis: Caching layer for email data and agent coordination
# psycopg2-binary: PostgreSQL adapter for Supabase connections
# pydantic: Data validation and settings management
# fastapi: Modern web framework for APIs
# uvicorn: ASGI server for FastAPI
